# MJ EATS

Backedn MJ Eats

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Entities

### User Model:

- id
- createdAt
- updatedAt

- email
- password
- role(client|owner|delivery)

### User Crud

- CreateAccount
- Login
- See Profile
- Edit Profile
- Verify Email
